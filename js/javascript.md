# String in JavaScript
group of more than one character is called string

### Declaration

```javascript
let name="naresh";

```
### Methods of String
* name.toUpperCase()
### output
```
NARESH
```
* name.toLowerCase()
### example

```javascript
let name=NARESH;
name.toLowerCase();
console.log(name);

```
output
```
naresh
```

* name.trim()
